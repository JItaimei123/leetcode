> 给定一个只包含 `'('`、`')'`、`'{'`、`'}'`、`'['`、`']'` 的字符串，判断字符串是否有效。
> 有效条件：

  1. 左括号必须用相同类型的右括号闭合。

  2. 左括号必须以正确的顺序闭合。

  3. 每个右括号都有一个对应的左括号。

示例：

- `"()[]{}"` → 有效

- `"([)]"` → 无效（顺序不对）

- `"{[]}"` → 有效

## 算法核心思想

使用 **栈（stack）** 后进先出的特性：

1. 遍历字符串的每一个字符。

2. 遇到 **左括号**（`(`, `[`, `{`）→ 压入栈。

3. 遇到 **右括号**（`)`, `]`, `}`）→

  - 如果栈为空，说明没有左括号可以匹配 → **无效**。

  - 否则弹出栈顶元素，检查是否与当前右括号匹配（例如 `')'` 必须对应 `'('`）。

  - 若不匹配 → **无效**。

4. 遍历结束后，栈为空表示所有左括号都被正确匹配 → **有效**；否则无效。

```C++
class Solution {
public:
    bool isValid(string s) {
        stack<char> st;//创建一个栈
        for (auto ch : s) {//遍历每一个字符
            if (ch == '[' || ch == '{' || ch == '(')
                st.push(ch);//将左括号压入栈
            else {
                if (st.empty())//栈为空，也就是没有左括号被压入
                    return false;
                char top = st.top();//将栈顶元素存在top变量中
                st.pop();//弹出栈顶元素

                if ((ch == '}' && top != '{') || (ch == ']' && top != '[') ||
                    (ch == ')' && top != '('))//判断是否匹配
                    return false;
            }
        }
        return st.empty();//栈空则全匹配
    }
};
```

## 深入探究

为什么要用

```C++
  if((ch == '}' && top != '{') ||(ch == ')' && top != '(') || (ch == ']' && top != '['))
            return false;
```

而不用

```C++
  if((ch == '}' && top == '{') ||(ch == ')' && top == '(') || (ch == ']' && top == '['))
            return true;
```

### 原因：当遇到右括号且栈顶匹配时，应该继续遍历，而不是返回 true。只有当所有字符处理完且栈为空才返回 true。
