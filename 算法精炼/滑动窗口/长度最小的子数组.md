## 209. 长度最小的子数组

给定一个含有 `n` 个正整数的数组和一个正整数 `target`。

找出该数组中满足其总和大于等于 `target` 的长度最小的 子数组 `[numsl, numsl+1, ..., numsr-1, numsr]`，并返回其长度。如果不存在符合条件的子数组，返回 `0`。

思路： 用滑动窗口双指针，因为这里的单调性，所以可以用滑动窗口。在这里的数组中，先设定两个指针，`left` 和 `right` 分别都放在起始位置，这里题目要求是总和大于等于 `target`，也就是让 `left` 和 `right` 区间求和的过程，所以具备单调性，len 越长，和一定越大，所以可以用同向双指针枚举，这是由暴力枚举优化而来的，总结成一种思想叫滑动窗口。

### 具体写法思路

定义双指针 `left = 0`，`right = 0`，初始化这个"窗口"。初始化完成之后定义一个 `sum` 来记录此时窗口的和，再定义一个 `ans` 变量初始化为一个比 `n` 还要大的值，这样后续用 `min` 函数记录更新后的子数组的长度时，可以通过 `ans` 的返回值判断到底 `while(sum ≥ target)` 到底有没有发生，也就是有没有符合的子数组。

此时的和就是 2，再让 `left` 不动，让 `right` 右移，此时 `right` 指向 3，那这个窗口此时的和就是 2+3（`sum += nums[right]`），判断与 7 的关系，＜7 则 `right` 继续右移，此时窗口和为 6 依旧＜7，继续右移，此时窗口的和为 8，符合条件，这个过程用 for 循环实现记录该子数组的长度，然后保持 `right` 不动，把 `left` 指向的元素移出窗口，此时窗口的和变为 `sum -= nums[left] = 6`，再把 `left` 向右移。此时再判断与 7 的关系，若＜则 `right` 继续向右移动，继续 `sum += nums[right]`，以此类推，符合条件就 `left` 右移，更新 `sum`，再判断，不符合条件就 `right` 右移，更新 `sum` 再判断，直到 `right` "出界"。

所以滑动窗口的过程很简单：先初始化窗口 → 同向移动双指针 → 更新窗口的值 → 进行判断 → 移动指针形成闭环。

示例 1：

```C++
输入：target = 7, nums = [2,3,1,2,4,3]
输出：2
解释：子数组 [4,3] 是该条件下的长度最小的子数组。
```

### 具体代码实现

```C++
class Solution {
public:
    int minSubArrayLen(int target, vector<int>& nums) 
    {
        int n = nums.size();
        int sum = 0;
        int left = 0;
        int right = 0;
        int ans = n + 1;
        while(right < n)
        {
            sum += nums[right];
            right++;
            while(sum >= target)
            {
                ans = min(ans,right - left);
                sum -= nums[left];
                left++;
            }
        }
        return ans == n+1 ? 0 : ans;
    }
};
```
