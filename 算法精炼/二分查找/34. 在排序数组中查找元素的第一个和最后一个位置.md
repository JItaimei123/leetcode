# [34. 在排序数组中查找元素的第一个和最后一个位置](https://leetcode.cn/problems/find-first-and-last-position-of-element-in-sorted-array/)

给你一个按照非递减顺序排列的整数数组 `nums`，和一个目标值 `target`。请你找出给定目标值在数组中的开始位置和结束位置。

如果数组中不存在目标值 `target`，返回 `[-1, -1]`。

你必须设计并实现时间复杂度为 `O(log n)` 的算法解决此问题。

## 示例

示例 1：

```C++
输入：nums = [5,7,7,8,8,10], target = 8
输出：[3,4]
```

## 题解（二分查找模板）

这是一个很好的模板练习题，直接套入查找左边界和查找右边界的模板即可

```C++
vector<int> searchRange(vector<int>& nums, int target) 
    {
        int right1 = nums.size()-1;
        int right2 = nums.size()-1;
        int left1 = 0;
        int left2 = 0;
        vector<int> res;
        if(nums.empty())
        return {-1,-1};
        while(left1 < right1)//查找左边界，左闭右开
        {
            int mid1 = left1 + (right1 - left1)/2;//左中点
            if(nums[mid1] >= target)
            right1 = mid1;//因为是左中点，所以这里不会陷入死循环
            else
            left1 = mid1 + 1;
        }
        if (left1 == nums.size() || nums[left1] != target) 
        {
            return {-1, -1};
        }
        res.push_back(left1);
        while(left2<right2)//查找右边界
        {
            int mid2 = left2 + (right2 - left2 + 1)/2;//右中点
            if(nums[mid2] <= target)
            left2 = mid2;//因为是右中点，所以这里不会陷入死循环
            else
            right2 = mid2 - 1;
        }
        res.push_back(left2);
        return res;
    }
```
