给你一个整数数组 `nums` ，除某个元素仅出现 **一次** 外，其余每个元素都恰出现 **三次 。**请你找出并返回那个只出现了一次的元素。

## 核心逻辑

假设只有**两种数字**：

- **目标数字 X**（出现 1 次）

- **其他数字**（每个都出现 3 次）

对于**某一位**（比如第 0 位）：

- 所有“其他数字”在这一位上，要么是 0，要么是 1。但不管它们是 0 还是 1，因为每个数字出现 3 次，所以它们贡献的 1 的总数 = 0 或 3 或 6... **一定是 3 的倍数**。

- 目标数字 X 在这一位上，要么是 0（贡献 0 个 1），要么是 1（贡献 1 个 1）。

**所以**：
总 1 的个数 = (3 的倍数) + (0 或 1)

| 总 1 的个数 % 3 | 结果 | 说明 |
| --- | --- | --- |
| = 0 | 目标数字这一位是 0 | 3的倍数 + 0 |
| = 1 | 目标数字这一位是 1 | 3的倍数 + 1 |
| = 2 | 不可能出现 | 因为目标数字最多贡献 1 |

```C++
class Solution {
public:
    int singleNumber(vector<int>& nums) 
    {
        int result = 0;
        for(int bit = 0;bit<32;bit++)//遍历int的32个二进制位
        {
            int bitcount = 0;
            for(auto ch :nums)//遍历每一个元素
            {
                if(ch >> bit &1)
            //将元素的补码右移bit，最末尾的一位即使一开始的第bit位，若为1则&1为1，if（1）为真，进入循环->bitcount++，若为0则跳过本次循环
                {
                    bitcount++;
                }
            }
            if(bitcount %3 !=0)//！=0时说明三个一样的数的第bit位都是1，否则为0
            {
                result |=(1<<bit);//将result的第bit位设置为1
            }
        }
        return result;
    }
};
```

**整体思路：**创建一个变量result接收目标数的每一位，遍历int的32个二进制位，创建一个变量bitcount统计1的出现次数，通过循环遍历每个元素，通过右移操作取到第bit位的二进制数，将其与1进行与运算（&），若为1则第bit位的二进制数为1，此时bitcount++，再根据如果是三个一样的数，那贡献的1的个数一定是3的倍数，所以有bitcount % 3，若 = 1则目标数的第bit位为1，同时将1向左移bit位，再用按位或运算符（|），这样就可以将第bit位二进制数设置为1。
