## [622.设置循环队列](#622设置循环队列)

设计你的循环队列实现。循环队列是一种线性数据结构，其操作表现基于 FIFO（先进先出）原则并且队尾被连接在队首之后以形成一个循环。

你的实现应该支持如下操作：

- `MyCircularQueue(k)`: 构造器，设置队列长度为 k。

- `Front`: 从队首获取元素。如果队列为空，返回 -1。

- `Rear`: 获取队尾元素。如果队列为空，返回 -1。

- `enQueue(value)`: 向循环队列插入一个元素。如果成功插入则返回真。

- `deQueue()`: 从循环队列中删除一个元素。如果成功删除则返回真。

- `isEmpty()`: 检查循环队列是否为空。

- `isFull()`: 检查循环队列是否已满。

```C++
class MyCircularQueue {
private:
    struct Node {
        int val;
        Node* next;
    };
    
    Node* head;
    Node* tail;
    int capacity;
    int count;

public:
    MyCircularQueue(int k) {
        head = nullptr;
        tail = nullptr;
        capacity = k;
        count = 0;
    }
    
    bool enQueue(int value) {
        if (isFull()) return false;
        Node* newNode = (Node*)malloc(sizeof(Node));
        newNode->val = value;
        newNode->next = nullptr;
        if (isEmpty()) {
            head = tail = newNode;
            tail->next = head;
        } else {
            tail->next = newNode;
            tail = newNode;
            tail->next = head;
        }
        count++;
        return true;
    }
    
    bool deQueue() {
        if (isEmpty()) return false;
        if (head == tail) {
            free(head);
            head = tail = nullptr;
        } else {
            Node* toDelete = head;
            head = head->next;
            tail->next = head;
            free(toDelete);
        }
        count--;
        return true;
    }
    
    int Front() {
        if (isEmpty()) return -1;
        return head->val;
    }
    
    int Rear() {
        if (isEmpty()) return -1;
        return tail->val;
    }
    
    bool isEmpty() { return count == 0; }
    bool isFull() { return count == capacity; }
};
```
