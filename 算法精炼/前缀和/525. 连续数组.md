给定一个二进制数组 `nums` , 找到含有相同数量的 `0` 和 `1` 的最长连续子数组，并返回该子数组的长度。

**示例 1：**

```C++
输入：nums = [0,1]
输出：2
说明：[0, 1] 是具有相同数量 0 和 1 的最长连续子数组。
```

**示例 2：**

```C++
输入：nums = [0,1,0]
输出：2
说明：[0, 1] (或 [1, 0]) 是具有相同数量 0 和 1 的最长连续子数组。
```

**示例 3：**

```C++
输入：nums = [0,1,1,1,1,1,0,0,0]
输出：6
解释：[1,1,1,0,0,0] 是具有相同数量 0 和 1 的最长连续子数组。
```

思路：题目要求0和1的数量一致，那就将0变为-1，让-1和1结合为0，这样只需要求和为0的最长子数组，依旧用前缀和思想，让这个前缀和为0，然后长度最短，即可求出和为0长度最长的子数组，这里哈希表中存的<int,int>分别是**和与下标**，所以**hash[0] = -1**

```C++
class Solution {
public:
    int findMaxLength(vector<int>& nums) 
    {
        int n = nums.size();
        unordered_map<int,int> hash;
        hash[0] = -1;
        int sum = 0,ans = 0;
        for(int i = 0;i<n;i++)
        {
            sum += (nums[i] == 0 ? -1 : 1);
            if(hash.count(sum))
            ans = max(ans,i-hash[sum]);
            else
            hash[sum] = i;
        }
        return ans;
    }
};
```
