给你两个单链表的头节点 `headA` 和 `headB` ，请你找出并返回两个单链表相交的起始节点。如果两个链表不存在相交节点，返回 `null` 。

图示两个链表在节点 `c1` 开始相交**：**

![160_statement.png](https://cdn2.flowus.cn/oss/07d61fad-4960-46fa-b3d0-058cd6eee635/160_statement.png?time=1784381400&token=9a3d92f5f327bed491c8de717787356fcfbb36e6def14a1b7424bf479ae14742&role=free)

题目数据 **保证** 整个链式结构中不存在环。

**注意**，函数返回结果后，链表必须 **保持其原始结构** 。

解题思路：

•遍历两个链表找到尾结点，同时求出两个链表的⻓度。尾结点相同则相交，尾结点不同则不相交。

•算出两个链表的⻓度差距，让长的链表先⾛差距步，两个链表再同时⾛，第⼀个相等的结点就是交

点。

![image.png](https://cdn2.flowus.cn/oss/ba277757-9c03-4f95-8331-2968521c83d7/image.png?time=1784381400&token=8128f41a9696040e983475d940a9f89f78b5e5151a3653943546f3c7b343aab5&role=free)

```C++
    ListNode *meetNode(ListNode *head1,ListNode *head2)
    if(!head1 || !head2)
    return nullptr;
    ListNode *cur1 = head1,*cur2 = head2;
    int len1 = 1,len2 = 1;
    while(cur1->next)
    {
        cur1 = cur1->next;
        len1++;
    }
    while(cur2->next)
    {
        cur2 = cur2->next;
        len2++;
    }
    if(cur1!=cur2)
    return NULL;
    int gap = abs(len1-len2);
    ListNode *longlist = (len1>len2)?head1:head2;
    ListNode *shortlist = (len1>len2)?head2:head1;
    while(gap--)
    {
         longlist = longlist->next;
    }
    while(longlist!=shortlist)
    {
        longlist = longlist->next;
        shortlist=  shortlist->next;
    }
    return shortlist;
```

**“让指针停在最后一个节点”** 意味着你能拿到尾节点地址，适合需要尾节点信息的场景（如判断链表相交）。
**“让指针继续往后走到 NULL”** 是常规遍历的终点，适合只关心节点值或计数的场景。

| 需求 | 推荐循环 | 原因 |
| --- | --- | --- |
| 获取尾节点、判断相交 | while(cur->next) | 需要尾节点地址 |
| 只计算长度 | 两者皆可 | 但 while(cur) 更直观（len 从0开始）而while（cur→next）的len从1开始 |
| 遍历所有节点并处理值 | while(cur) | 最常用，简单直观 |
| 在最后一个节点处停止，并对其做特殊操作 | while(cur->next) | 指针停在了尾节点上 |
