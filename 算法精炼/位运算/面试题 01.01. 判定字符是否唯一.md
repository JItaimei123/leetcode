实现一个算法，确定一个字符串 `s` 的所有字符是否全都不同。

**示例 1：**

```C++
输入: s = "leetcode"
输出: false 
```

**示例 2：**

```C++
输入: s = "abc"
输出: true
```

**限制：**

- `0 <= len(s) <= 100`

- `s[i]`仅包含小写字母

- 如果你不使用额外的数据结构，会很加分。

## 解法一：哈希表

```C++
class Solution {
public:
    bool isUnique(string astr) 
    {
        unordered_set<char> hash;
        for(auto c : astr)
        {
            if(hash.count(c))
            return false;
            hash.insert(c);
        }
        return true;
    }
};
```

## 解法二：借助位图思想用一个 `int` 的 26 个 bit 来标记哪些字母出现过

思路：建立一个位图，利用ASCII码**`int i = c - 'a'`**映射每个字符的关系，判断位图中这一位是不是1，若是1代表出现过，return false，若不是1就把这一位改成1

```C++
class Solution {
public:
    bool isUnique(string astr) 
    {
        if(astr.size() > 26)
        return false;
        int bitmap = 0;
        for(auto ch : astr)
        {
            int i = ch - 'a';//映射字母与下标关系
            if(((bitmap >> i) & 1) == 1)//位图中已存在
            return false;
            bitmap |= 1 << i;
        }   
        return true;
    }
};
```
