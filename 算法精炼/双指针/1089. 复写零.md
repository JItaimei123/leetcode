给你一个长度固定的整数数组 `arr` ，请你将该数组中出现的每个零都复写一遍，并将其余的元素向右平移。

注意：请不要在超过该数组长度的位置写入元素。请对输入的数组 **就地** 进行上述修改，不要从函数返回任何东西。

### 思路1：先用双指针确定最后一个会被复写的元素，再从后往前进行复写操作，如果从前往后的话，后面的数会被覆盖掉，这里不需要删除元素，所以从前往后不适用，这里采用从后往前的遍历方式

先找到最后一个会被复写的元素，用slow遍历要复写的元素，用fast遍历复写后的元素，若是遇到0则fast+=2，slow++，否则slow++，fast++，最后还需要slow- - ，fast- -，因为当fast指向下标n-1的时候，也就是指向最后一个数的时候，还是会进入循环然后slow++ fast++，最终到fast指向n的时候才终止循环，这时候需要同步回退才能让slow真正指向最后一个复写的数，主要还是因为这个终止条件

对于边界情况的处理：当slow遍历到的最后一个复写元素是0的话

```C++
class Solution {
public:
    void duplicateZeros(vector<int>& arr) {
        int n = arr.size();
        int slow = 0, fast = 0;
        while (fast < n) {
            if (arr[slow] == 0)
                fast += 2;
            else
            fast++;
            slow++;
        }
        slow--;
        fast--;
        if (fast == n) {
            arr[n - 1] = 0;//在 fast -= 2;的时候会跳过n-1位置，所以需要提前放好
            slow--;
            fast -= 2;
        }
        while (fast >= 0) {
            if (arr[slow] == 0) {
                arr[fast] = 0;
                fast--;
                arr[fast] = 0;
            } else {
                arr[fast] = arr[slow];
            }
            slow--;
            fast--;
        }
    }
};
```

### 思路2：用c++stl容器额外开辟空间解决

```C++
class Solution {
public:
    void duplicateZeros(vector<int>& arr) 
    {
        vector<int> res;
        for(auto x : arr)
        {
        res.push_back(x);
        if(x == 0)
        res.push_back(0);    
        }
        for(int i = 0;i<arr.size();i++)
        {
            arr[i] = res[i];
        }
    }
};
```
