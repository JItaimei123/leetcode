# [0～n-1 中缺失的数字](https://leetcode.cn/problems/que-shi-de-shu-zi-lcof/)

一个长度为 n-1 的递增排序数组中的所有数字都是唯一的，并且每个数字都在范围 0～n-1 之内。在范围 0～n-1 内的 n 个数字中有且只有一个数字不在该数组中，请找出这个数字。

## 示例

**示例 1：**

```C++
输入: [0,1,3]
输出: 2
```

**示例 2：**

```C++
输入: [0,1,2,3,4,5,6,7,9]
输出: 8
```

## 题解（二分查找 · 二段性思想）

### 思路

有序数组 + 查找唯一缺失的数字，适合用二分查找。

**关键性质：** 在不缺失的情况下，每个位置的 `nums[i] == i`。缺失之后，后面的元素都会比下标大 1，即 `nums[i] > i`。

所以数组被分成两段：
- **左段：** `nums[i] == i` ✅
- **右段：** `nums[i] > i`（实际是 `nums[i] == i+1`）❌

要找的就是 **第一个 `nums[i] != i` 的位置**（左边界），或者 **最后一个 `nums[i] == i` 的位置 + 1**（右边界）。

---

### 左边界查找风格（左中点版本）

> 找第一个 `nums[i] != i` 的位置 → 条件 `nums[mid] == mid` 成立说明左段还没结束，收缩左边界

```C++
class Solution {
public:
    int missingNumber(vector<int>& nums) {
        int left = 0, right = nums.size();
        while (left < right) {
            int mid = left + (right - left) / 2;  // 左中点
            if (nums[mid] == mid)   // 前 mid 个都没缺失 → 去右半找
                left = mid + 1;
            else                    // mid 处已错位 → 去左半找
                right = mid;
        }
        return left;
    }
};
```

**执行过程** `[0,1,3]`：
| left | right | mid | nums[mid] | 动作 |
|------|-------|-----|-----------|------|
| 0 | 3 | 1 | 1 | `==mid` → `left=2` |
| 2 | 3 | 2 | 3 | `!=mid` → `right=2` |
| 2 | 2 | 退出 | | **返回 2** ✅ |

---

### 右边界查找风格（右中点版本）

> 找最后一个 `nums[i] == i` 的位置 → 条件成立时收缩左边界，答案 = 该位置 + 1

```C++
class Solution {
public:
    int missingNumber(vector<int>& nums) {
        int left = 0, right = nums.size() - 1;
        while (left < right) {
            int mid = left + (right - left + 1) / 2;  // 右中点
            if (nums[mid] == mid)   // 当前位置没缺失，最后一个满足的在右边
                left = mid;
            else                    // 当前位置已错位，去左边找
                right = mid - 1;
        }
        // 循环结束后 left 是最后一个 nums[i]==i 的位置
        // 缺失值 = left + 1；若连第 0 位都错位了，说明缺失 0
        return nums[left] == left ? left + 1 : left;
    }
};
```

**执行过程** `[0,1,3]`：
| left | right | mid | nums[mid] | 动作 |
|------|-------|-----|-----------|------|
| 0 | 2 | 1 | 1 | `==mid` → `left=1` |
| 1 | 2 | 2 | 3 | `!=mid` → `right=1` |
| 1 | 1 | 退出 | | `nums[1]==1` → 返回 `1+1=2` ✅ |

---

### 对比总结

| 风格 | 中点公式 | 条件成立时 | 条件不成立时 | 最终结果 |
|------|---------|-----------|------------|---------|
| 🔵 **左边界查找** | `(l+r)/2` | `l = mid+1` | `r = mid` | `return left`（直接就是答案） |
| 🔴 **右边界查找** | `(l+r+1)/2` | `l = mid` | `r = mid-1` | `return left + 1` 或特殊处理 |
