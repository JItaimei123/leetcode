## 1658. 将 x 减到 0 的最小操作数

给你一个整数数组 `nums` 和一个整数 `x`。每一次操作时，你应当移除数组 `nums` 最左边或最右边的元素，然后从 `x` 中减去该元素的值。请注意，需要 **修改** 数组以供接下来的操作使用。

如果可以将 `x` **恰好** 减到 `0`，返回 **最小操作数**；否则，返回 `-1`。

**示例 1：**

```C++
输入：nums = [1,1,4,2,3], x = 5
输出：2
解释：最佳解决方案是移除后两个元素，将 x 减到 0 。
```

**示例 2：**

```C++
输入：nums = [5,6,7,8,9], x = 4
输出：-1
```

**思路：** 正面做的话这题会很难，既要算左边又要算右边，不妨用**正难则反**的思想，用滑动窗口维护一个除了左边和右边和分别是 `a+b = x` 的窗口，这个中间窗口维护的值应该是 `sum - x`，要求最小的操作数，就让中间这个窗口的长度达到最长即可。

所以问题就转化为：找出一个最长的子数组，值为 `sum - x`。

```C++
class Solution {
public:
    int minOperations(vector<int>& nums, int x) 
    {
        int n = nums.size();
        int right = 0;
        int left = 0;
        int sum = 0;        // 窗口目前和的值
        int total = 0;
        for(auto num : nums)
        {
            total += num;   // 整个数组的值
        }   
        if(total < x)
        return -1;
        int maxlen = -1;
        int target = total - x; // 窗口应该维护的定值
        while(right < n)
        {
            sum += nums[right];
            while(sum > target && left <= right)
            {
                sum -= nums[left];
                left++;
            }
            if(sum == target)
            maxlen = max(maxlen,right-left+1);
            right++;
        }
        return maxlen == -1 ? -1 : n-maxlen;
    }
};
```
