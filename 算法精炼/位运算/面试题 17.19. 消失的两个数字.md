给定一个数组，包含从 1 到 N 所有的整数，但其中缺了两个数字。你能在 O(N) 时间内只用 O(1) 的空间找到它们吗？

以任意顺序返回这两个数字均可。

**示例 1：**

```C++
输入：[1]
输出：[2,3]
```

**示例 2：**

```C++
输入：[2,3]
输出：[1,4]
```

**思路：**用给定数组和完整数组相比，可知完整数字比给定数组多了两个数字a和b，所以这两个数字就是只出现一次的数字

**解法：**

1.将两个数组异或，得出的结果是a^b

2.找到a与b“差异”的一位，a与b的第`diff`位一个是0一个是1 也就是`((tmp >> diff) & 1) == 1`

3.根据这个差异分组，将a和b分别分到两个组中

### 分组规则

- 第 `diff` 位为 `1` 的 → 分到 `group1`：给定数组和完整数组中第`diff`位为1的数字构成的数组

- 第 `diff` 位为 `0` 的 → 分到 `group0`：给定数组和完整数组中第`diff`位为0的数字构成的数组

再分别与a b 异或，也就转化为分两组实现“只出现一次的数字”

```C++
class Solution {
public:
    vector<int> missingTwo(vector<int>& nums) 
    {
        //1.异或完整数组和给定数组得出a^b
        int tmp = 0;
        for(auto num : nums)
        tmp ^= num;
        for(int i = 1;i<=nums.size()+2;i++)
        tmp ^= i;
        //2.找差异出现在第几位
        int diff = 0;
        while(((tmp >> diff) & 1) == 0)
        {
            diff++;
        }
        //3.利用这个差异分组实现“只出现一次的数字”
        int a = 0,b = 0;
        for(auto num : nums)
        if(((num >> diff) & 1) == 1)  
        a ^= num;
        else
        b ^= num;
        for(int i = 1 ;i <= nums.size()+2;i++)
        if(((i >> diff) & 1) == 1)
        a ^= i;
        else
        b ^= i;
        return {a,b};
    }
};
```
