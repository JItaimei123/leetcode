## [206.反转链表](#206反转链表)

给你单链表的头节点 `head`，请你反转链表，并返回反转后的链表。

**示例 1：**

```C++
输入：head = [1,2,3,4,5]
输出：[5,4,3,2,1]
```

**示例 2：**

```C++
输入：head = [1,2]
输出：[2,1]
```

**示例 3：**

```C++
输入：head = []
输出：[]
```

```C++
class Solution {
public:
    ListNode* reverseList(ListNode* head) 
    {
        struct ListNode * cur = head;
        struct ListNode *newhead = NULL;
        while(cur)
        {
            struct ListNode *next = cur->next;  // 保存下一个节点
            cur->next = newhead;                 // 将该节点指向新链表的表头
            newhead = cur;                       // 更新链表表头为当前节点
            cur = next;                          // cur 移到下一个节点
        }
        return newhead;
    }
};
```
