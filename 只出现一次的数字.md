给你一个整数数组 `nums` ，除某个元素仅出现 **一次** 外，其余每个元素都恰出现 **三次 。**请你找出并返回那个只出现了一次的元素。

你必须设计并实现线性时间复杂度的算法且使用常数级空间来解决此问题。

**示例 1：**

```C++
输入：nums = [2,2,3,2]
输出：3
```

**示例 2：**

```C++
输入：nums = [0,1,0,1,0,1,99]
输出：99
```

思路：一个数有32个比特位，所以`for(int i = 0;i<32;i++`)作为外循环，再通过`for(auto x : nums)`遍历数组中的每一个数，要想统计第i位上的1的个数，先x >> i ,再让其余的bit位都是0，然后再&1即可判断，&1的同是也会把其余为都清零，因为1的二进制是00000......1，用sum记录统计的1的个数，只可能是`3n`或`3n+1`，再用**`sum %=  3`**即可判断1的个数，从而可得知这个只出现一次的数的第i位是1还是0，如果是1的话就通过`ret |= 1 << i`把第i位改成1

```C++
class Solution {
public:
    int singleNumber(vector<int>& nums) 
    {
        int ret = 0;
        for(int i = 0;i<32;i++)
        {
            int sum = 0;
            for(auto x : nums)
            {
                if((x >> i) & 1 == 1)
                sum++;
            }
            sum %= 3;
            if(sum == 1) ret |= 1 << i;
        }    
        return ret;
    }
};
```
