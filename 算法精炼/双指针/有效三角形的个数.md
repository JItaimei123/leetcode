## 611. 有效三角形的个数

给定一个包含非负整数的数组 `nums` ，返回其中可以组成三角形三条边的三元组个数。

示例 1:

```C++
输入: nums = [2,2,3,4]
输出: 3
解释: 有效的组合是: 
2,3,4 (使用第一个 2)
2,3,4 (使用第二个 2)
2,2,3
```

示例 2:

```C++
输入: nums = [4,2,3,4]
输出: 4
```

### 思路

现将数组升序排序，利用双指针。固定最大边 `nums[i]`（从后往前遍历），将 left 放在开头位置，将 right 放在 i-1 位置。若此时 `nums[left] + nums[right] > nums[i]`，那就证明 `right - left` 这个区间的组合都满足三角形的要求（因为 left 往右两边之和会不断变大），所以此时 `count += (right - left)`，然后 right--；否则 left++ 将小的值变大再试。以此类推，直到 left ≥ right 时终止循环，返回 count。

### 代码

```C++
class Solution {
public:
    int triangleNumber(vector<int>& nums) 
    {
        sort(nums.begin(), nums.end());
        int n = nums.size();
        int count = 0;
        for (int i = n - 1; i >= 2; i--)
        {   
            int left = 0;
            int right = i - 1;
            while (left < right)
            {
                if (nums[left] + nums[right] > nums[i])
                {
                    count += (right - left);
                    right--;
                }
                else
                {
                    left++;
                }
            }
        }    
        return count;
    }
};
```

> 注意：外层循环不能写成 `for(int i = n - 1; i >= 1; i--)` 配合 `right = i`，虽然表达的意思一样，但在后续判断代码中 i 和 n-1 与 right 会产生歧义导致编译错误，需要区分开。
