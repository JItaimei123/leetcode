给定一个单词列表 `words` 和一个整数 `k` ，返回前 `k` **个出现次数最多的单词。

返回的答案应该按单词出现频率由高到低排序。如果不同的单词有相同出现频率， **按字典顺序** 排序。

```C++
class Solution {
public:
    vector<string> topKFrequent(vector<string>& words, int k) {
        // 1. 统计每个单词出现的次数
        map<string, int> countMap;
        for (const auto& word : words) {
            countMap[word]++;
        }

        // 2. 把 map 里的键值对存到 vector 中，方便排序
        vector<pair<string, int>> vec;
        for (const auto& p : countMap) {
            vec.push_back({p.first, p.second});
        }

        // 3. 排序：频率高的在前；频率相同，字典序小的在前
        sort(vec.begin(), vec.end(), [](const auto& a, const auto& b) {
          // a和b的类型会自动推导为需要比较的元素的类型（pair<string,int>）  
          if (a.second != b.second) {
                return a.second > b.second;  // 频率降序
            }
            return a.first < b.first;        // 字典序升序
        });

        // 4. 取前 k 个单词
        vector<string> result;
        for (int i = 0; i < k; ++i) {
            result.push_back(vec[i].first);
        }
        return result;
    }
};
```
