## [相交链表](#相交链表)

给你两个单链表的头节点 `headA` 和 `headB`，请你找出并返回两个单链表相交的起始节点。如果两个链表不存在相交节点，返回 `null`。

解题思路：

- 遍历两个链表找到尾结点，同时求出两个链表的长度。尾结点相同则相交，尾结点不同则不相交。

- 算出两个链表的长度差距，让长的链表先走差距步，两个链表再同时走，第一个相等的结点就是交点。

```C++
ListNode *meetNode(ListNode *head1, ListNode *head2) {
    if (!head1 || !head2) return nullptr;
    ListNode *cur1 = head1, *cur2 = head2;
    int len1 = 1, len2 = 1;
    while (cur1->next) {
        cur1 = cur1->next;
        len1++;
    }
    while (cur2->next) {
        cur2 = cur2->next;
        len2++;
    }
    if (cur1 != cur2) return NULL;
    int gap = abs(len1 - len2);
    ListNode *longlist = (len1 > len2) ? head1 : head2;
    ListNode *shortlist = (len1 > len2) ? head2 : head1;
    while (gap--) {
        longlist = longlist->next;
    }
    while (longlist != shortlist) {
        longlist = longlist->next;
        shortlist = shortlist->next;
    }
    return shortlist;
}
```

### while(cur) vs while(cur->next)

|需求|推荐循环|原因|
|-|-|-|
|获取尾节点、判断相交|`while(cur->next)`|需要尾节点地址|
|只计算长度|两者皆可|但 `while(cur)` 更直观（len 从0开始）而 `while(cur->next)` 的 len 从1开始|
|遍历所有节点并处理值|`while(cur)`|最常用，简单直观|
|在最后一个节点处停止，并对其做特殊操作|`while(cur->next)`|指针停在了尾节点上|
