## [删除有序数组中的重复项2](#删除有序数组中的重复项2)

给你一个有序数组 `nums`，请你**原地**删除重复出现的元素，使得出现次数超过两次的元素**只出现两次**，返回删除后数组的新长度。

```C++
class Solution {
public:
    int removeDuplicates(vector<int>& nums) 
    {
        if(nums.size() <= 2)
            return nums.size();
        int k = 2;
        for(int i = 2; i < nums.size(); i++)
        {
            if(nums[i] != nums[k-2])
            {
                nums[k++] = nums[i];
            }
        }
        return k;
    }
};
```

### 通用解法：删除有序数组中的n个重复项

```C++
int removeDuplicates(vector<int>& nums, int maxRepeat = 2) {
    if (nums.size() <= maxRepeat) return nums.size();
    int k = maxRepeat;
    for (int i = maxRepeat; i < nums.size(); i++) {
        if (nums[i] != nums[k - maxRepeat]) {
            nums[k++] = nums[i];
        }
    }
    return k;
}
```
