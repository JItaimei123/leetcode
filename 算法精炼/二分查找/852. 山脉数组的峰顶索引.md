# [852. 山脉数组的峰顶索引](https://leetcode.cn/problems/peak-index-in-a-mountain-array/)

给定一个长度为 `n` 的整数 **山脉** 数组 `arr` ，其中的值递增到一个 **峰值元素** 然后递减。

返回峰值元素的下标。

你必须设计并实现时间复杂度为 `O(log(n))` 的解决方案。

## 示例

**示例 1：**

```C++
输入：arr = [0,1,0]
输出：1
```

**示例 2：**

```C++
输入：arr = [0,2,1,0]
输出：1
```

**示例 3：**

```C++
输入：arr = [0,10,5,2]
输出：1
```

## 题解（二分查找 · 二段性思想）

思路：用二分算法的思想，但不是套模板，依旧是利用 **二段性** 来解答。根据比较 `arr[mid]` 与相邻元素的结果，判断峰顶在左还是右，从而缩小区间。

### 左边界查找风格（左中点版本）

> 当 **下降** 时 `arr[mid] > arr[mid+1]`，峰顶在 mid 或左边，收缩右边界 → 左边界查找

```C++
class Solution {
public:
    int peakIndexInMountainArray(vector<int>& arr) 
    {
        int left = 0,right = arr.size();
        while(left < right)
        {
            int mid = left + (right - left)/2;
            if(arr[mid] > arr[mid+1])//下降阶段 → 收缩右边界
            right = mid;
            else//上升阶段 → 收缩左边界
            left = mid + 1;
        }
        return left;
    }
};
```

### 右边界查找风格（右中点版本）

> 当 **上升** 时 `arr[mid] > arr[mid-1]`，峰顶在 mid 或右边，收缩左边界 → 右边界查找

```C++
class Solution {
public:
    int peakIndexInMountainArray(vector<int>& arr) 
    {
        int left = 0,right = arr.size() - 1;
        while(left < right)
        {
            int mid = left + (right - left + 1)/2;
            if(arr[mid] > arr[mid-1])//上升阶段 → 收缩左边界
            left = mid;
            else//下降阶段 → 收缩右边界
            right = mid - 1;
        }
        return left;
    }
};
```
