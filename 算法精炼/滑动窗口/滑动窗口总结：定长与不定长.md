|对比维度|定长窗口（30.串联所有单词的字串）|不定长 - 求最长 (如 3. 无重复字符)|不定长 - 求最短 (如 76. 最小覆盖子串)|
|-|-|-|-|
|收缩判断|窗口长度 `> k`|窗口不合法 (`while(重复)`)|窗口合法 (`while(已覆盖)`)|
|收缩目的|维持固定大小|排除重复 / 重新合法化|缩小窗口，寻找更短解|
|更新答案时机|收缩后|收缩后 (此时窗口刚好合法)|收缩前 (此时窗口仍合法)|
|更新答案位置|`if (长度 == k)` 内|`while` 收缩循环之后|`while` 收缩循环之内，`left` 移动之前|
|收缩次数|`if`，只执行一次|`while`，持续移动直到合法|`while`，持续移动直到不合法|
|移动单位|可能是字符或单词|通常按字符|按字符|
|窗口状态|大小固定|动态变化 (合法)|动态变化 (收缩中)|

代码结构对比
定长滑动窗口（以 30. 串联所有单词的子串 为例）

```C++
class Solution {
public:
    vector<int> findSubstring(string s, vector<string>& words) 
    {
        vector<int> res;
        unordered_map<string,int> hash1;//用于统计words中出现的单词的次数
        for(auto ch : words)
        {
            hash1[ch]++;
        }    
        int wordlen = words[0].size();
        int totallen = wordlen * words.size();
        for(int i = 0;i<wordlen;i++)//执行wordlen次
        {
            unordered_map<string,int> hash2;//执行wordlen次每个窗口的单词出现的频次
            for(int left = i,right = i,count = 0;right + wordlen <= s.size();right += wordlen)
            {
                //进窗口 + 维护count
                string in = s.substr(right,wordlen);
                hash2[in]++;
                if(hash1.count(in) && hash2[in] <= hash1[in])//words中可能有重复的单词
                count++;
                //判断是否出窗口
                if(right-left+1 > totallen)
                {
                    //出窗口
                    string out = s.substr(left,wordlen);
                    if(hash1.count(out) && hash2[out] <= hash1[out])
                    count--;
                    hash2[out]--;
                    left += wordlen;
                }
                //更新结果
                if(count == words.size())
                res.push_back(left);
            }
        }
        return res;
    }
};
```

定长滑动窗口（以[30. 串联所有单词的子串](https://leetcode.cn/problems/substring-with-concatenation-of-all-words/)为例）

特点：

* 收缩条件基于窗口长度（`right - left + 1 > totalLen`）
* 每次收缩只移出一个单位
* 收缩是为了保持窗口固定大小

不定长滑动窗口
（以 [76. 最小覆盖子串](https://leetcode.cn/problems/minimum-window-substring/) 与 [3. 无重复字符的最长子串](https://leetcode.cn/problems/longest-substring-without-repeating-characters/)为例）

```C++
class Solution {
public:
    string minWindow(string s, string t) 
    {
        unordered_map<char,int> hash1;//用于统计t中的字符的频次
        unordered_map<char,int> hash2;//用于统计s中的字符的频次
        for(auto ch : t)
        hash1[ch]++;
        int minlen = INT_MAX;
        int start = 0;
        for(int right = 0,left = 0,count = 0;right < s.size();right++)
        {
            //进窗口+维护count
            hash2[s[right]]++;
            if(hash1.count(s[right]) && hash2[s[right]] <= hash1[s[right]])
            count++;
            //判断出窗口条件
            while(count == t.size())            
            //出窗口
            {
                //出窗口之前更新结果
                if(right-left+1<minlen)
                {
                minlen = right-left+1;
                start =  left;
                }
                if(hash1.count(s[left]) && hash2[s[left]] <= hash1[s[left]])
                count--;
                hash2[s[left]]--;
                left++;
            }
        }
        return minlen == INT_MAX ? "" : s.substr(start,minlen);
    }
};
```

```C++
class Solution {
public:
    int lengthOfLongestSubstring(string s) 
    {
        int ans = INT_MIN;
        unordered_map<char,int> hash;
        for(int right = 0,left = 0;right<s.size();right++)
        {
            //进窗口
            hash[s[right]]++;
            //判断窗口是否合法：进窗口的是否是重复字符
            while(hash[s[right]] > 1)
            {
                //出窗口
                hash[s[left]]--;
                if(hash[s[left]] == 0)
                hash.erase(s[left]);
                left++;
            }
            //更新结果
            ans = max(ans,right-left+1);
        }
        return ans == INT_MIN ? 0 : ans;
    }
};
```

特点：

* 收缩条件基于窗口是否合法（`valid == need.size()`）
* 收缩是持续性的（用 `while` 不断移动 `left`）
* 收缩是为了寻找最短/最长解

一句话总结
定长窗口：长度决定何时收缩，收缩是为了维持固定大小。 不定长窗口：合法性决定何时收缩，收缩是为了寻找更优解。

总结模板：
定长滑动窗口
进窗口→判断出窗口条件：达到定长时→出窗口→更新结果

不定长滑动窗口（求最短）
进窗口→判断出窗口条件：窗口合法时→更新结果→出窗口

不定长滑动窗口（求最长）
进窗口→判断出窗口条件：窗口合法时→出窗口→更新结果

只有不定长滑动窗口（求最短）是在出窗口前更新结果
