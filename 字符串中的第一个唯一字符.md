给定一个字符串 `s` ，找到它的第一个不重复的字符，并返回它的索引 。如果不存在，则返回 `-1`

```C++
class Solution {
public:
    int firstUniqChar(string s) {
        int count[26] = {0};//用一个计数数组来统计每一个字符出现了几次
        for(char & c : s)
        {
            count[c-'a']++;//用ASCII码值来判定一个字母出现了几次
  //c-'a'=0,则此时的字母为a，同理c-'a'=1，则为b，这样就能统计每个出现在这个数组中的字母
        }
        for(int i = 0; i<s.size();i++)
        {
            if(count[s[i]-'a']==1)//s[i]-'a'就是代表了某一个字母的ASCII码值
            {
            return i;
            }
        }
     return -1;
    }
    
};
```

这里用的就是哈希思想中的**直接定值法**

这里，哈希函数就是 `c - 'a'`，把字符 `'a'` 映射到下标 `0`，`'b'` 映射到 `1`，以此类推。这是一个**完美的一一对应**，没有冲突。

如果键是 ASCII 码，但你想从 `0` 开始存：

```C++
// 假设要统计 ASCII 码 32 ~ 126 的字符出现次数
int count[95] = {0};
for (char &c : s) {
    count[c - 32]++;  // 线性映射：地址 = 键 - 32
}
```

| ASCII 码 | 地址 = ASCII - 32 |
| --- | --- |
| 32（空格） | 0 |
| 65（'A'） | 33 |
| 122（'z'） | 90 |
