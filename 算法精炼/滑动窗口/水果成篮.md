## 904. 水果成篮

你正在探访一家农场，农场从左到右种植了一排果树。这些树用一个整数数组 `fruits` 表示，其中 `fruits[i]` 是第 `i` 棵树上的水果 种类。

你想要尽可能多地收集水果。然而，农场的主人设定了一些严格的规矩，你必须按照要求采摘水果：

- 你只有 两个 篮子，并且每个篮子只能装 单一类型 的水果。每个篮子能够装的水果总量没有限制。

- 你可以选择任意一棵树开始采摘，你必须从 每棵 树（包括开始采摘的树）上 恰好摘一个水果。采摘的水果应当符合篮子中的水果类型。每采摘一次，你将会向右移动到下一棵树，并继续采摘。

- 一旦你走到某棵树前，但水果不符合篮子的水果类型，那么就必须停止采摘。

给你一个整数数组 `fruits`，返回你可以收集的水果的 最大 数目。

示例 1：

```C++
输入：fruits = [1,2,1]
输出：3
解释：可以采摘全部 3 棵树。
```

示例 2：

```C++
输入：fruits = [0,1,2,2]
输出：3
解释：可以采摘 [1,2,2] 这三棵树。
如果从第一棵树开始采摘，则只能采摘 [0,1] 这两棵树。
```

思路： 这题本质就是要找到一个最长的连续子数组，使其包含不超过两种不同的数字。这里涉及到统计频率以及按照要求动态维护子数组，所以用哈希表 + 滑动窗口解决。滑动窗口的本质就是用双指针更高效地遍历符合要求的子串，本质上就是一种更高效的暴力。

```C++
class Solution {
public:
    int totalFruit(vector<int>& fruits) 
    {
        int right = 0;
        int left = 0;
        int ans = -1;
        int n = fruits.size();
        unordered_map<int,int> count; // 哈希表统计每种水果出现的次数
        while(right < n)
        {
            count[fruits[right]]++;
            while(count.size() > 2) // 维护一个只有两种水果，但长度最长的子串
            {
                count[fruits[left]]--;
                if(count[fruits[left]] == 0)
                count.erase(fruits[left]); // 为了更新哈希表中的"种类"，不删除的话哈希表内的种类就不止2种了
                left++;
            }
            ans = max(ans,right-left+1);
            right++;
        }
        return ans;
    }
};
