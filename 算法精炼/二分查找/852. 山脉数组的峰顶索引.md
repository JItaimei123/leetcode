给定一个长度为 `n` 的整数 **山脉** 数组 `arr` ，其中的值递增到一个 **峰值元素** 然后递减。

返回峰值元素的下标。

你必须设计并实现时间复杂度为 `O(log(n))` 的解决方案。

**示例 1：**

```C++
输入：arr = [0,1,0]
输出：1
```

**示例 2：**

```C++
输入：arr = [0,2,1,0]
输出：1
```

**示例 3：**

```C++
输入：arr = [0,10,5,2]
输出：1
```

思路：这里用二分算法的思想，但不是套模板，依旧是利用二段性来解答，当山峰处于下降阶段的时候，峰顶一定在mid或者mid左边，可以类比为**查��左边界**

```C++
class Solution {
public:
    int peakIndexInMountainArray(vector<int>& arr) 
    {
        int left = 0,right = arr.size();
        while(left < right)
        {
            int mid = left + (right - left)/2;
            if(arr[mid] >= arr[mid+1])//下降阶段，峰值在 mid 或更左边，去左边找->更新right
            right = mid;
            else//上升阶段，要去右边找->更新left
            left = mid + 1;
        }
        return left;
    }
};
```

### 左边界查找风格（左中点版本）

> 当 **下降** 时 `arr[mid] > arr[mid+1]`，峰顶在 mid 或左边，收缩右边界 → 左边界查找

```C++
class Solution {
public:
    int peakIndexInMountainArray(vector<int>& arr) 
    {
        int left = 0,right = arr.size();
        while(left < right)
        {
            int mid = left + (right - left)/2;
            if(arr[mid] > arr[mid+1])//下降阶段 → 收缩右边界去左边找
            right = mid;
            else//上升阶段 → 收缩左边界去右边找
            left = mid + 1;
        }
        return left;
    }
};
```

### 右边界查找风格（右中点版本）

> 当 **上升** 时 `arr[mid] > arr[mid-1]`，峰顶在 mid 或右边，收缩左边界 → 右边界查找

```C++
class Solution {
public:
    int peakIndexInMountainArray(vector<int>& arr) 
    {
        int left = 0,right = arr.size() - 1;
        while(left < right)
        {
            int mid = left + (right - left + 1)/2;
            if(arr[mid] > arr[mid-1])//上升阶段 → 收缩左边界去右边找
            left = mid;
            else//下降阶段 → 收缩右边界去左边找
            right = mid - 1;
        }
        return left;
    }
};
```
