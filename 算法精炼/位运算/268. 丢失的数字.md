给定一个包含 `[0, n]` 中 `n` 个数的数组 `nums` ，找出 `[0, n]` 这个范围内没有出现在数组中的那个数。

**示例 1：**

**输入：**nums = [3,0,1]

**输出：**2

**解释：**`n = 3`，因为有 3 个数字，所以所有的数字都在范围 `[0,3]` 内。2 是丢失的数字，因为它没有出现在 `nums` 中。

**示例 2：**

**输入：**nums = [0,1]

**输出：**2

**解释：**`n = 2`，因为有 2 个数字，所以所有的数字都在范围 `[0,2]` 内。2 是丢失的数字，因为它没有出现在 `nums` 中。

**示例 3：**

**输入：**nums = [9,6,4,2,3,5,7,0,1]

**输出：**8

**解释：**`n = 9`，因为有 9 个数字，所以所有的数字都在范围 `[0,9]` 内。8 是丢失的数字，因为它没有出现在 `nums` 中。

## 解法1:哈希表

```C++
class Solution {
public:
    int missingNumber(vector<int>& nums) 
    {
        unordered_set<int> hash;
        for(auto num : nums)
        {
            hash.insert(num);
        }
        for(int i = 0;i<=nums.size();i++)
        {
            if(!hash.count(i))
            return i;
        }
        return -1;
    }
};
```

## 解法2：二分查找

```C++
class Solution {
public:
    int missingNumber(vector<int>& nums) 
    {
        sort(nums.begin(),nums.end());
        int left = 0,right = nums.size();
        while(left < right)
        {
            int mid = left + (right - left) / 2;
            if(nums[mid] == mid)//左边没缺失
            left = mid + 1;
            else
            right = mid;
        }    
        return left;
    }
};
```

## 解法3：异或操作

思路：根据异或的运算律可知，`a ^ a =  0`,`a ^ 0 = a`,`a ^ b ^ c = a ^ (b ^ c)`

### 准备工作

`nums = [1, 2, 3, 4]`，`n = 4`，完整序列是 `0, 1, 2, 3, 4`。

异或的规则只有两条：

1. 两个相同的数异或得 0：`a ^ a = 0`

2. 0 和任何数异或得那个数本身：`0 ^ a = a`

---

## 总结：

方法就是**用完整的数组 ^已给的数组**，这里给的数组是`nums``1`` = [1, 2, 3, 4]`，完整数组是`nums2 = [0,1,2,3,4]`,这里就是在用`nums1 ^ nums2`,其中1234都是重复的，根据`a ^ a = 0`与`a ^ 0 = a`可知剩下的就是`0 ^ (a ^ a) = 0`

---

### 结果

`ans = 0`，就是缺失的数。

---

### 为什么结果是 0？

完整序列是 `0, 1, 2, 3, 4`，数组是 `1, 2, 3, 4`。两个序列中，`1, 2, 3, 4` 都出现了两次，异或后全部抵消成 0，只剩下完整序列中的 `0` 没有配对，所以结果是 `0`。

---

```C++
class Solution {
public:
    int missingNumber(vector<int>& nums) {
        int n = nums.size();
        int ans = 0;
        for (int i = 0; i <= n; i++)
            ans ^= i; // 完整序列异或
        for (int num : nums)
            ans ^= num; // 给定数组异或
        return ans;
    }
};
```

先让完整数组全部异或一遍，再与给定数组异或
