## [232.用栈实现队列](#232用栈实现队列)

请你仅使用两个栈实现先入先出队列。队列应当支持一般队列支持的所有操作（`push`、`pop`、`peek`、`empty`）。

### 整体思路

队列的特点是**先进先出（FIFO）**，而栈的特点是**后进先出（LIFO）**。用两个栈来模拟队列，核心思想是：

- **一个栈专门负责"入队"**（称为 `inStack`）

- **另一个栈专门负责"出队"和"取队首"**（称为 `outStack`）

入队时，直接把元素压入 `inStack`。
出队或取队首时，如果 `outStack` 为空，就把 `inStack` 里的所有元素**依次弹出并压入 `outStack`**。
这样，`outStack` 的栈顶就是原来 `inStack` 的栈底，也就是队列的**队首元素**。

每个元素最多会被从 `inStack` 移到 `outStack` 一次，因此 `pop` 和 `peek` 的**均摊时间复杂度为 O(1)**。

```C++
class MyQueue {
private:
    stack<int> instack;
    stack<int> outstack;

public:
    MyQueue() {}

    void push(int x) { 
        instack.push(x); 
    }

    int pop() {
        if (outstack.empty()) {
            while (!instack.empty()) {
                outstack.push(instack.top());
                instack.pop();
            }
        }
        int topval = outstack.top();
        outstack.pop();
        return topval;
    }

    int peek() {
        if (outstack.empty()) {
            while (!instack.empty()) {
                outstack.push(instack.top());
                instack.pop();
            }
        }
        return outstack.top();
    }

    bool empty() { 
        return instack.empty() && outstack.empty(); 
    }
};
```
