## 76. 最小覆盖子串

给定两个字符串 `s` 和 `t`，长度分别是 `m` 和 `n`，返回 `s` 中的 最短窗口 子串，使得该子串包含 `t` 中的每一个字符（包括重复字符）。如果没有这样的子串，返回空字符串 `""`。

测试用例保证答案唯一。

示例 1：

```C++
输入：s = "ADOBECODEBANC", t = "ABC"
输出："BANC"
解释：最小覆盖子串 "BANC" 包含来自字符串 t 的 'A'、'B' 和 'C'。
```

思路：

不定长滑动窗口求最小子串 + 哈希表。当窗口合法（已覆盖 `t` 中所有字符）时开始收缩左边界，并且 一定要在收缩前更新结果。

```C++
class Solution {
public:
    string minWindow(string s, string t) 
    {
        unordered_map<char,int> hash1;//用于统计t中的字符的频次
        unordered_map<char,int> hash2;//用于统计s中的字符的频次
        for(auto ch : t)
        hash1[ch]++;
        int minlen = INT_MAX;
        int start = 0;
        for(int right = 0,left = 0,count = 0;right < s.size();right++)
        {
            //进窗口+维护count
            hash2[s[right]]++;
            if(hash1.count(s[right]) && hash2[s[right]] <= hash1[s[right]])
            count++;
            //判断出窗口条件：窗口合法时
            while(count == t.size())            
            //出窗口
            {
                //出窗口之前更新结果
                if(right-left+1<minlen)
                {
                minlen = right-left+1;
                start =  left;
                }
                if(hash1.count(s[left]) && hash2[s[left]] <= hash1[s[left]])
                count--;
                hash2[s[left]]--;
                left++;
            }
        }
        return minlen == INT_MAX ? "" : s.substr(start,minlen);
    }
};
```
