有一个自行车手打算进行一场公路骑行，这条路线总共由 `n + 1` 个不同海拔的点组成。自行车手从海拔为 `0` 的点 `0` 开始骑行。

给你一个长度为 `n` 的整数数组 `gain` ，其中 `gain[i]` 是点 `i` 和点 `i + 1` 的 **净海拔高度差**（`0 <= i < n`）。请你返回 **最高点的海拔** 。

```C++
class Solution {
public:
    int largestAltitude(vector<int>& gain) 
    {
        int cur = 0;//当前海拔
        int res = 0;//最高海拔
        for(auto x : gain)
        {
            cur += x;//cur 的值不断更新为当前所在位置的海拔
            res = max(res,cur);
        }
        return res;
    }
};
//海拔[i + 1] = 海拔[i] + gain[i]
```
