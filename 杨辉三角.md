给定一个非负整数 *`numRows`，*生成「杨辉三角」的前 *`numRows`* 行。

在**「杨辉三角」**中，每个数是它左上方和右上方的数的和。

![1626927345-DZmfxB-PascalTriangleAnimated2.gif](https://cdn2.flowus.cn/oss/fc397845-15da-44e6-97e3-e37309c92846/1626927345-DZmfxB-PascalTriangleAnimated2.gif?time=1784380500&token=fc97274c9c940d415a839cdcb53c92af557e10688c514a2164769a61a6f09c48&role=free)

**示例 1:**

```C++
输入: numRows = 5
输出: [[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]
```

**示例 2:**

```C++
输入: numRows = 1
输出: [[1]]
```

核心思想：找出杨辉三角的规律，发现每一行头尾都是1，将所有数初始化为0，再分别将头尾改为1，或者直接将所有数初始化为1也可以，这样会更简洁一点中间第[j]个数等于上一行[j-1]+

[j]

```C++
class Solution {
public:
    vector<vector<int>> generate(int numRows) 
    {
        vector<vector<int>> vv(numRows);
        for(int i = 0; i < numRows;i++)
        {
            vv[i].resize(i+1,0);
            vv[i].front() = vv[i].back() = 1;
          //可以用 vv[i].resize(i+1,1);替代，会更简洁一些
          //vv[i][0] = vv[i][vv[i].size()-1] = 1;
        }
        for(int i = 2;i<numRows;i++)//从第三行开始遍历每一行
        {
            for(int j = 1;j<vv[i].size()-1;j++)//从第二行开始遍历每一列，j=1就是跳过第0列
            {
                vv[i][j] = vv[i-1][j] + vv[i-1][j-1];
            }
        }
        return vv;
    }
};
```
