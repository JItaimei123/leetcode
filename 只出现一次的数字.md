给你一个 **非空** 整数数组 `nums` ，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

**示例 1 ：**

**输入：**nums = [2,2,1]

**输出：**1

**示例 2 ：**

**输入：**nums = [4,1,2,1,2]

**输出：**4

**示例 3 ：**

**输入：**nums = [1]

**输出：**1

## 思路：用异或运算，

## 根本原因：异或运算的“自抵消”特性

**核心规则就两条：**

1. **一个数异或自己 = 0**（`a ^ a = 0`）

2. **一个数异或 0 = 它自己**（`a ^ 0 = a`）

再加上异或满足 **交换律**（顺序可以随便换），所以整个数组异或一遍，效果是：

### 假设数组为[4,1,2,1,2]

数组：`[4, 1, 2, 1, 2]`

1. `0 ^ 4 = 4`

2. `4 ^ 1 = 5`

3. `5 ^ 2 = 7`

4. `7 ^ 1 = 6` → 这里相当于 `(4 ^ 1 ^ 2) ^ 1 = 4 ^ 2 ^ (1 ^ 1) = 4 ^ 2 ^ 0 = 4 ^ 2 = 6`

5. `6 ^ 2 = 4` → 相当于 `(4 ^ 2) ^ 2 = 4 ^ (2 ^ 2) = 4 ^ 0 = 4`

看到没：

- 两个 `1` 抵消成 `0`

- 两个 `2` 抵消成 `0`

- 最后只剩下 `4`

## 换一种更直观的理解方式

把数组里的数字按任意顺序做异或，效果是一样的。
我们可以刻意“把成对的放一起”：

`(1 ^ 1) ^ (2 ^ 2) ^ 4 = 0 ^ 0 ^ 4 = 4`

因为异或满足交换律，不管数组怎么乱序，最终结果都和上面一样。

> **成对出现的数相互抵消成 0，剩下的就是那个“落单”的数。**

```C++
class Solution {
public:
    int singleNumber(vector<int>& nums) 
    {
        int value = 0;
        for(auto ch : nums)
        {
            value ^= ch;
        }
        return value;
    }
};
```
