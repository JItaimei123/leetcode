## [移除链表元素](#移除链表元素)

给你一个链表的头节点 `head` 和一个整数 `val`，请你删除链表中所有满足 `Node.val == val` 的节点，并返回**新的头节点**。

**示例 1：**

```C++
输入：head = [1,2,6,3,4,5,6], val = 6
输出：[1,2,3,4,5]
```

**示例 2：**

```C++
输入：head = [], val = 1
输出：[]
```

**示例 3：**

```C++
输入：head = [7,7,7,7], val = 7
输出：[]
```

解题思路：

- 准备一前一后两个指针 prev 和 cur，遍历链表，当 cur->val 等于要删除的 val 时，prev 指向 cur 的下一个结点更改链接关系，释放 cur 结点

- 在没有哨兵位的链表中，如果第一个结点的值等于 val，则会导致 prev 为空，要单独判断处理更新头指针；这里可以选择增加哨兵位结点，避免 prev 为空的情况。
