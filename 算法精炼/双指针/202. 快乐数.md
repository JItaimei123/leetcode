编写一个算法来判断一个数 `n` 是不是快乐数。

**「快乐数」** 定义为：

- 对于一个正整数，每一次将该数替换为它每个位置上的数字的平方和。

- 然后重复这个过程直到这个数变为 1，也可能是 **无限循环** 但始终变不到 1。

- 如果这个过程 **结果为** 1，那么这个数就是快乐数。

如果 `n` 是 *快乐数* 就返回 `true` ；不是，则返回 `false` 。

**示例 1：**

```C++
输入：n = 19
输出：true
解释：
12 + 92 = 82
82 + 22 = 68
62 + 82 = 100
12 + 02 + 02 = 1
```

### 思路：这里对示例1和示例2画图分析：可知都是会陷入循环，都可以归为成环类，只是环里面的数字不同，如果是快乐数，那还环里的数都是1，所以这里定义**快慢指针**来判断在环中两个指针相遇时的数是不是1。注意这里是用数来充当指针，以这里的示例二为例子，slow初始为2，fast初始为2的下一位(bitsum(n))，因为循环的条件是while(slow ≠ fast)，若是一开始slow和fast都为2那就无法进入循环了

代码实现：先定义一个bitsum函数，用于计算这个数组中的下一位，再利用快慢指针的特性，让slow一次走一步，fast一次走两步，往后走通过bitsum函数控制，因为这里的指针是一个数，通过这个函数也可以更新bitsum函数中的n的值，直到两指针相遇，判断值是否为1

```C++
class Solution {
public:
    int bitsum(int n)
    {
      int sum = 0;
      while(n > 0)
      {
        int t = n % 10;
        sum += t * t;
        n /= 10;
      }
      return sum;
    }
    bool isHappy(int n) 
    {
        int slow = n,fast = bitsum(n);
        while(slow != fast)
        {
            slow = bitsum(slow);
            fast = bitsum(bitsum(fast));
        }
        return slow == 1;
    }
};
```
