# 一维前缀和

### 用处：快速得出数组中某个连续区间的和

### 解题步骤：

预处理一个前缀和数组，与原数组大小一致

假设原有数组arr内有元素{1,4,7,2,5,8,3,6,9}

        **索引index**为      1,2,3,4,5,6,7,8,9

        **前缀和数组dp**为{1, 5, 12, 14, 19, 27, 30, 36, 45}
dp[i] = dp[i-1] + arr[i] = ∑(k=1 到 i) arr[k]

### 根据以上特性 要求[l,r]区间的和，只需要让`dp[r] - dp[l-1]`

### 为什么索引要从1开始？

若i从0开始`dp[r] - dp[l-1]`会访问`dp[-1]`显然越界了，那就需要特殊讨论

若是从1开始要求[1,2]区间和的时候只需要求`dp[2] - dp[0]`,此时让`dp[0] = 0`即可

```C++
#include <iostream>
#include <vector>
using namespace std;

int main() {
    // 1. 读入数据
    int n, q;
    cin >> n >> q;
    vector<int> arr(n + 1);  // 索引从 1 开始
    for (int i = 1; i <= n; i++) {
        cin >> arr[i];
    }

    // 2. 预处理前缀和数组
    vector<long long> dp(n + 1, 0);//防止溢出
    for (int i = 1; i <= n; i++) {
        dp[i] = dp[i - 1] + arr[i];
    }

    // 3. 处理查询
    while (q--) {
        int l, r;
        cin >> l >> r;
        cout << dp[r] - dp[l - 1] << endl;
    }

    return 0;
}
```
