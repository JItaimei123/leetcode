给定一个包含非负整数的数组 `nums` ，返回其中可以组成三角形三条边的三元组个数。

**示例 1:**

```C++
输入: nums = [2,2,3,4]
输出: 3
解释:有效的组合是: 
2,3,4 (使用第一个 2)
2,3,4 (使用第二个 2)
2,2,3
```

**示例 2:**

```C++
输入: nums = [4,2,3,4]
输出: 4
```

思路：现将数组升序排序，利用双指针，将left放在开头位置，将right放在倒数第二个位置，若此时**num[left] + nums[right] > nums[i]**那就证明right - left这个区间的组合都满足三角形的要求，因为left往右两边之和会不断变大，所以此时改变两边之和中那条长的边→right- -，**否则**left++将小的值变大再试，以此类推知直到eft ≥ right的时候终止循环返回count

```C++
class Solution {
public:
    int triangleNumber(vector<int>& nums) 
    {
        sort(nums.begin(),nums.end());
        int n = nums.size();
        int count = 0;
        for(int i = n - 1;i>=2;i--)
        {   
            int left = 0;
            int right = i - 1;
            while(left < right)
            {
                if(nums[left] + nums[right] > nums[i])
                {
                    count += (right - left);
                    right--;
                }
                else
                {
                    left++;
                }
            }
        }    
        return count;
```

注意这里的写法

```C++
 for(int i = n - 1;i>=2;i--)
        {   
            int left = 0;
            int right = i - 1;
```
