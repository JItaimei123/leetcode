## [只出现一次的数2](#只出现一次的数2)

给你一个整数数组 `nums`，除某个元素仅出现**一次**外，其余每个元素都恰出现**三次**。请你找出并返回那个只出现了一次的元素。

**整体思路：** 创建一个变量 result 接收目标数的每一位，遍历 int 的32个二进制位，创建一个变量 bitcount 统计1的出现次数。通过循环遍历每个元素，通过右移操作取到第 bit 位的二进制数，将其与1进行与运算（&），若为1则第 bit 位的二进制数为1，此时 bitcount++。再根据如果是三个一样的数，那贡献的1的个数一定是3的倍数，所以有 bitcount % 3，若 = 1 则目标数的第 bit 位为1。

### 核心逻辑

对于某一位（比如第 0 位）：

- 所有"其他数字"在这一位上，贡献的 1 的总数 = 0 或 3 或 6... **一定是 3 的倍数**。

- 目标数字 X 在这一位上，要么是 0（贡献 0 个 1），要么是 1（贡献 1 个 1）。

|总 1 的个数 % 3|结果|说明|
|-|-|-|
|= 0|目标数字这一位是 0|3的倍数 + 0|
|= 1|目标数字这一位是 1|3的倍数 + 1|
|= 2|不可能出现|因为目标数字最多贡献 1|

```C++
class Solution {
public:
    int singleNumber(vector<int>& nums) 
    {
        int result = 0;
        for(int bit = 0; bit < 32; bit++)
        {
            int bitcount = 0;
            for(auto ch : nums)
            {
                if(ch >> bit & 1)
                {
                    bitcount++;
                }
            }
            if(bitcount % 3 != 0)
            {
                result |= (1 << bit);
            }
        }
        return result;
    }
};
```
