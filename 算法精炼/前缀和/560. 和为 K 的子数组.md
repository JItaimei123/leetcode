给你一个整数数组 `nums` 和一个整数 `k` ，请你统计并返回 *该数组中和为 `k` ****的子数组的个数* 。

子数组是数组中元素的连续非空序列。

**示例 1：**

```C++
输入：nums = [1,1,1], k = 2
输出：2
```

**示例 2：**

```C++
输入：nums = [1,2,3], k = 3
输出：2
```

### 思路：

![image.png](./images/image.png)

如上图所示，要求和为k的子数组，由前缀和思想可以转化为求值为`sum[i] - k`的子数组，**连续子数组的和 = 两个前缀和的差**再利用哈希表来存储前缀和的值以及每个值出现的次数。对于3的情况，是应对当[-1,0]这个区间不存在值的情况，提前放入一个0，与k匹配，否则无法匹配会漏掉数。

```C++
class Solution {
public:
    int subarraySum(vector<int>& nums, int k) 
    {
        unordered_map<int,int> hash;
        int sum = 0,ans = 0;
        hash[0] = 1;
        for(auto num:nums)
        {
        sum += num;//前缀和
        if(hash.count(sum-k))
        ans += hash[sum-k];
        hash[sum]++;//将每个前缀和记录在哈希表中，同时统计次数
        }
        return ans;
    }
};
```
