# [852. 山脉数组的峰顶索引](https://leetcode.cn/problems/peak-index-in-a-mountain-array/)

给定一个长度为 `n` 的整数 **山脉** 数组 `arr` ，其中的值递增到一个 **峰值元素** 然后递减。

返回峰值元素的下标。

你必须设计并实现时间复杂度为 `O(log(n))` 的解决方案。

## 示例

**示例 1：**

```C++
输入：arr = [0,1,0]
输出：1
```

**示例 2：**

```C++
输入：arr = [0,2,1,0]
输出：1
```

**示例 3：**

```C++
输入：arr = [0,10,5,2]
输出：1
```

## 题解（二分查找 · 二段性思想）

思路：用二分算法的思想，但不是套模板，依旧是利用 **二段性** 来解答。当山峰处于下降阶段的时候，峰顶一定在 mid 或者 mid 左边，可以类比为 **查找左边界**。

```C++
class Solution {
public:
    int peakIndexInMountainArray(vector<int>& arr) 
    {
        int left = 0,right = arr.size();
        while(left < right)
        {
            int mid = left + (right - left)/2;
            if(arr[mid] > arr[mid+1])//下降阶段，峰值在 mid 或更左边，去左边找->更新right
            right = mid;
            else//上升阶段，要去右边找->更新left
            left = mid + 1;
        }
        return left;
    }
};
```
