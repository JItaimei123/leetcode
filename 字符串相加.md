给定两个字符串形式的非负整数 `num1` 和`num2` ，计算它们的和并同样以字符串形式返回。

你不能使用任何內建的用于处理大整数的库（比如 `BigInteger`）， 也不能直接将输入的字符串转换为整数形式。

**示例 1：**

```C++
输入：num1 = "11", num2 = "123"
输出："134"
```

**示例 2：**

```C++
输入：num1 = "456", num2 = "77"
输出："533"
```

**示例 3：**

```C++
输入：num1 = "0", num2 = "0"
输出："0"
```

```C++
class Solution 
{
public:
    string addStrings(string num1, string num2) 
    {
        string str;//这里创建了一个容纳字符的容器，将计算的结果放入其中
        int end1 = num1.size() - 1;
        int end2 = num2.size() - 1;
        int next = 0;//记录进位
        while (end1 >= 0 || end2 >= 0) 
        {
            int val1 = end1 >= 0 ? num1[end1--] - '0' : 0;
            int val2 = end2 >= 0 ? num2[end2--] - '0' : 0;
            int ret = val1 + val2 + next;
            next = ret / 10;
            ret = ret % 10;//若ret
            str.insert(str.begin(), '0' + ret);
        }
        if (next == 1)//考虑1+9这种特殊情况
        {
            str.insert(str.begin(), '1');
        }
        return str;
    }
};
```

## 小知识点解剖：

```C++
 next = ret / 10;
 ret = ret % 10;
```

这里顺序不能反，先进位再取余

```C++
int val1 = end1 >= 0 ? num1[end1--] - '0' : 0;
int val2 = end2 >= 0 ? num2[end2--] - '0' : 0;
```

`num2[end2--] - '0'` 中的 `- '0'` 是为了将数字字符（如 `'5'`）转换为对应的整数值（如 `5`）。

- `num2[end2--]` 是一个字符，例如 `'5'`。

- `'0'` 也是一个字符，其 ASCII 值为 48。

- 两者相减，得到的是字符的 ASCII 码差值，因为数字字符的 ASCII 码是连续递增的，所以 `'5' - '0'` = 53 - 48 = 5。

如果不减 `'0'`，直接使用字符 `'5'` 参与算术运算，它会被提升为整数 53，导致计算结果错误。因此，`- '0'` 是字符到整数转换的常用技巧。

```C++
 str.insert(str.begin(), '0' + ret);
```

这里也是同理

```C++
if (next == 1)//考虑1+9这种特殊情况
```

这里不能写成if(ret≤10)，虽然最直白的原因确实是这个，但是**`ret` 的作用域仅限于循环内部**

在这里意味着**end1 >= 0 || end2 >= 0**为假

## 整体思路：

用end1与end2作为索引先遍历两个“数组”，**只要还有至少一个数字有未处理的位（索引≥0）就继续**，记录从低位到高位对齐的每一个数的相加后的余数与进位数，定义一个string容器存放这一串字符串，由于计算是从低位写起，所有要用到头插操作。

```C++
 class Solution {
public:
    string addStrings(string num1, string num2) {
        string str;
        int end1 = num1.size() - 1;
        int end2 = num2.size() - 1;
        int next = 0;
        while (end1 >= 0 || end2 >= 0) {
            int val1 = (end1 >= 0) ? num1[end1--] - '0' : 0;
            int val2 = (end2 >= 0) ? num2[end2--] - '0' : 0;
            int sum = val1 + val2 + next;
            next = sum / 10;
            int cur = sum % 10;
            str.push_back('0' + cur);   // 尾插
        }
        if (next == 1) {
            str.push_back('1');
        }
        reverse(str.begin(), str.end()); // 反转
        return str;
    }
};
```

注意这里尾插操作 `str.push_back('1');`可用`str +='1';`替代

对于**追加单个字符**，`push_back`、`operator+=` 和 `append(1, ch)` 的效果**完全相同**。它们都是在字符串末尾添加一个字符。

---

### 示例

cpp

```C++
string s;
char ch = 'a';

s.push_back(ch);        // s = "a"
s += ch;                // s = "aa"
s.append(1, ch);        // s = "aaa"
```

这三种写法最终结果相同。

---

### 细微差别

- `push_back`：只接受单个字符（`char` 类型）。

- `operator+=`：可以接受单个字符、字符串、`const char*` 等，功能更丰富。

- `append`：功能最强大，可以追加重复字符、子串、迭代器区间等，但对于单个字符，需要指定计数 `1`。
