## [移除元素](#移除元素)

思路1：类似顺序表删除，从前往后遍历，遇到等于val的值，则把后面的元素往前挪动覆盖。这样最坏情况下，大多数数据都等于val，时间复杂度就为 O(N²)

```C
#include<stdio.h>
int removeElement(int* nums, int numsSize, int val)
{
    for (int i = 0; i < numsSize; i++)
    {
        if (val == nums[i])
        {
            for (int j = i; j < numsSize - 1; j++)
            {
                nums[j] = nums[j + 1];
            }
            numsSize--;
        }
    }
    return numsSize;
}
int main() {
    int i = 0;
    int nums[] = { 3, 2, 2, 3 };
    int numsSize = sizeof(nums) / sizeof(nums[0]);
    int val = 3;
    int newsize = removeElement(nums, numsSize, 3);
    printf("%d ", newsize);
    return 0;
}
```

思路2：创建一个一样大小的数组tmp，遍历原数组把不等于val的元素依次挪动到tmp，再把数据拷贝回到原数组，这样时间复杂度就为 O(N)，但是空间复杂度也是 O(N)

```C
#include<stdio.h>
#include<stdlib.h>
int removeElement(int* nums, int numsSize, int val)
{
    int* tmp = (int*)malloc(numsSize * sizeof(int));
    if(tmp ==NULL)
        return 1;
    int k = 0;
    for (int i = 0; i < numsSize; i++)
    {
        if (nums[i] != val)
        {
            tmp[k++] = nums[i];
        }
    }
    for (int i = 0; i < k; i++) 
    {
        nums[i] = tmp[i];
    }
    free(tmp);
    return k;
}
int main() {
    int i = 0;
    int nums[] = { 3, 2, 2, 3 };
    int numsSize = sizeof(nums) / sizeof(nums[0]);
    int val = 3;
    int newsize = removeElement(nums, numsSize, val);
    printf("%d ", newsize);
    return 0;
}
```
