## [轮转数组](#轮转数组)

给定一个整数数组 `nums`，将数组中的元素向右轮转 `k` 个位置，其中 `k` 是非负数。

**示例 1:**

```

输入: nums = [1,2,3,4,5,6,7], k = 3

输出: [5,6,7,1,2,3,4]

解释:

向右轮转 1 步: [7,1,2,3,4,5,6]

向右轮转 2 步: [6,7,1,2,3,4,5]

向右轮转 3 步: [5,6,7,1,2,3,4]

```C++

**示例 2:**

```

输入：nums = [-1,-100,3,99], k = 2

输出：[3,99,-1,-100]

解释:

向右轮转 1 步: [99,-1,-100,3]

向右轮转 2 步: [3,99,-1,-100]

```C++

### 解法一：逐次轮转（O(N²)，可能超时）

我们采用每次将数组中数据右轮转一位，轮转k次的思路，则时间复杂度为O(N²)，空间复杂度为O(1)。需要注意的是，LeetCode这道题目有时间要求，这个算法思路无法达到要求，所以无法AC。

```c
#include<stdio.h>
void rotate(int* nums, int numsSize, int k)
{
    k %= numsSize;
    if (k == 0)
        return;
    while (k--)
    {
        int tmp = nums[numsSize - 1];
        for (int i = numsSize - 1; i > 0; i--)
        {
            nums[i] = nums[i - 1];
        }
        nums[0] = tmp;
    }
}
int main()
{
    int nums[] = { 1,2,3,4,5 };
    int numsSize = sizeof(nums) / sizeof(nums[0]);
    int k = 3;
    rotate(nums, numsSize, k);
    for (int i = 0; i < numsSize; i++)
    {
        printf("%d ", nums[i]);
    }
    printf("\n");
    return 0;
}
```

### 解法二：空间换时间（O(N)）

我们采用新的思路**空间换时间**，开辟一个额外的跟nums一样大的临时数组tmp，将nums的后k个数据挪动到tmp的前k个位置，再将nums前n-k个挪动到tmp的后n-k个位置，再将tmp数组中的值拷贝回到nums数组，则时间复杂度为 T(N) = O(N)，空间复杂度为S(N) = O(N)

```C
#include<stdio.h>
void rotate(int* nums, int numsSize, int k)
{
    k %= numsSize;
    if (k == 0)
        return;
    int tmp[3];
    int j = 0;
    for (int i = numsSize - k; i < numsSize; i++)
    {
        tmp[j++] = nums[i];
    }
    for (int i = numsSize - k - 1; i >= 0; i--)
    {
        nums[i + k] = nums[i];
    }
    for (int i = 0; i < k; i++)
    {
        nums[i] = tmp[i];
    }
}
int main()
{
    int nums[] = { 1,2,3,4,5,6,7,8 };
    int numsSize = sizeof(nums) / sizeof(nums[0]);
    int k = 3;
    rotate(nums, numsSize, k);
    for (int i = 0; i < numsSize; i++)
    {
        printf("%d ", nums[i]);
    }
    printf("\n");
    return 0;
}
```
