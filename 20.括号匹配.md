## [20.括号匹配](#20括号匹配)

给定一个只包含 `'('`、`')'`、`'{'`、`'}'`、`'['`、`']'` 的字符串，判断字符串是否有效。

有效条件：

1. 左括号必须用相同类型的右括号闭合。

2. 左括号必须以正确的顺序闭合。

3. 每个右括号都有一个对应的左括号。

示例：

- `"()[]{}"` → 有效

- `"([)]"` → 无效（顺序不对）

- `"{[]}"` → 有效

### 算法核心思想

使用**栈（stack）**后进先出的特性：

1. 遍历字符串的每一个字符。

2. 遇到**左括号**（`(`, `[`, `{`）→ 压入栈。

3. 遇到**右括号**（`)`, `]`, `}`）→ 如果栈为空则无效；否则弹出栈顶元素检查匹配。

4. 遍历结束后，栈为空表示所有左括号都被正确匹配。

```C++
class Solution {
public:
    bool isValid(string s) {
        stack<char> st;
        for (auto ch : s) {
            if (ch == '[' || ch == '{' || ch == '(')
                st.push(ch);
            else {
                if (st.empty())
                    return false;
                char top = st.top();
                st.pop();
                if ((ch == '}' && top != '{') || 
                    (ch == ']' && top != '[') ||
                    (ch == ')' && top != '('))
                    return false;
            }
        }
        return st.empty();
    }
};
```

### 深入探究

为什么要用：

```C++
if((ch == '}' && top != '{') || ...) return false;
```

而不用：

```C++
if((ch == '}' && top == '{') || ...) return true;
```

**原因：** 当遇到右括号且栈顶匹配时，应该继续遍历，而不是返回 true。只有当所有字符处理完且栈为空才返回 true。
