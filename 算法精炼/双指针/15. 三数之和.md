给你一个整数数组 `nums` ，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j`、`i != k` 且 `j != k` ，同时还满足 `nums[i] + nums[j] + nums[k] == 0` 。请你返回所有和为 `0` 且不重复的三元组。

**注意：**答案中不可以包含重复的三元组。

**示例 1：**

```C++
输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]
解释：
nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0 。
nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0 。
nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0 。
不同的三元组是 [-1,0,1] 和 [-1,-1,2] 。
注意，输出的顺序和三元组的顺序并不重要。
```

**解法思路如上图所示**，我在这里也叙述一下，现有一个数组要求三数之和为定值，求这三个数有多少种组合，要去重，不能由相同的三个数组成，先固定一个数的位置，转化为利用双指针求两数之和为定值，过程中同时进行去重操作，去重操作对双指针部分的数生效，同时也对固定的那个数生效，以图中的数组为例子现固定-4，在[-4，6]这个区间用双指针找定值，这里先找到-1和5，先找-1后面有没有重复的，只需要看相邻的即可，因为这里已经排序过了，显然没有，所以执行left++和right--，然后再找到一组是0和4，这时候就找到重复的了，检测到0重复就left++，检测到4重复就right--，直到不重复，然后再进行定值判断，不符合就继续用条件判断让left++，right--遍历，直到left ≥ right 停止。

```C++
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        sort(nums.begin(), nums.end());
        vector<vector<int>> result;//返回的是“多个三元组”，每个三元组本身又是“三个整数”。
        int n = nums.size();

        for (int i = 0; i < n - 2; ++i) {
            // 去重：跳过重复的 i
            if (i > 0 && nums[i] == nums[i - 1]) continue;

            int target = -nums[i];
            int left = i + 1;
            int right = n - 1;

            while (left < right) {
                int sum = nums[left] + nums[right];
                if (sum == target) {
                    result.push_back({nums[i], nums[left], nums[right]});
                    // 去重：跳过重复的 left 和 right
                    while (left < right && nums[left] == nums[left + 1]) left++;
                    while (left < right && nums[right] == nums[right - 1]) right--;
                    left++;
                    right--;
                } else if (sum < target) {
                    left++;
                } else {
                    right--;
                }
            }
        }
        return result;
    }
};
```
