给你一个非负整数 `x` ，计算并返回 `x` 的 **算术平方根** 。

由于返回类型是整数，结果只保留 **整数部分** ，小数部分将被 **舍去 。**

**注意：**不允许使用任何内置指数函数和算符，例如 `pow(x, 0.5)` 或者 `x ** 0.5` 。

**示例 1：**

```C++
输入：x = 4
输出：2
```

**示例 2：**

```C++
输入：x = 8
输出：2
解释：8 的算术平方根是 2.82842..., 由于返回类型是整数，小数部分将被舍去。
```

思路：假设target=17

现有数组[1,2,3,4,5,6,7]根据target=17可以将数组中的平方数二分化，因为这里要将小数舍去，所以是向下取整，若x^2 > target,则x^2后面的数不可能是答案，反之x^2≤target的话，前面的数可能是答案，也有可能直接命中target，所以<和=的情况归类在一起，所以这里是**查找右边界**的情况

```C++
class Solution {
public:
    int mySqrt(int x) 
    {
        int left = 1;
        int right = x;
        if(x == 0) return 0;
        while(left<right)
        {
            int mid = left+(right-left+1)/2;
            if(mid * mid <= x)
            {
            left = mid;
            }
            else
            right = mid-1;
        }
        return left;
    }
};
```
